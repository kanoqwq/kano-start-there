### 食用方法

**本地调试**

```bash
git clone https://github.com/kanoqwq/kano-start-there.git
cd kano-start-there
npm i
npm run dev
```

**打包编译**

```bash
npm run build
```

*打包好的文件在`./dist`目录下*
